KeyboardAvoidingScrollView class

Embeds the child in a SingleChildScrollView wrapped with a KeyboardAvoidingContainer. If the child contains a focused widget such as a TextField, it will auto-scroll so that that it is just visible above the keyboard, plus any additional bottomPadding.
